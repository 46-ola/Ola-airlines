# ✈️ Ola Airlines — Pioneering the Skies

**Interactive aviation education platform inspired by the Wright Brothers' legacy — featuring historical documents, an aviation knowledge quiz, and a student's aeronautical journey.**

Ola Airlines is a single-page web platform celebrating the pioneering spirit of the Wright Brothers and the history of flight. Built with vanilla HTML, CSS, and JavaScript, it combines educational content with engaging interactivity — designed as an educational hub for future aerospace engineers.

---

## 🚀 Features

- **Wright Brothers' Documents** — browse original excerpts from historical letters, patent filings, and diary entries (1900–1903), presented in styled cards with a "read full document" modal view.
- **Aviation Knowledge Quiz** — an 18-question interactive quiz on aviation history, with instant scoring, per-question feedback, and a retake option.
- **My Aeronautical Journey** — a personal profile section introducing the student behind the project and their path toward aerospace engineering.
- **Typing effect** — the hero title types itself out letter by letter on page load.
- **Scroll reveal animations** — section titles and cards fade in smoothly as you scroll down the page.
- **Responsive mobile menu** — an animated hamburger menu for a clean navigation experience on phones and tablets.
- **Custom visual identity** — navy-and-gold color palette, smooth transitions, and a polished, airline-inspired design.

---

## 🛠️ Built With

- **HTML5** — semantic page structure
- **CSS3** — custom styling, responsive design, animations (no framework)
- **Vanilla JavaScript** — DOM manipulation, quiz logic, IntersectionObserver-based scroll animations, mobile menu toggle
- **Google Fonts** — Inter & Space Grotesk
- **Font Awesome 6** — iconography

No build tools, frameworks, or dependencies required — the entire site runs from a single static HTML file.

## 💻 Getting Started

1. Clone the repository:
   ```bash

   ```
2. Open `airline.html` in your browser.

That's it — no server, no build step, no dependencies.
